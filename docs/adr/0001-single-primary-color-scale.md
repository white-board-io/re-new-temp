# Single primary color scale containing all three brand hexes

The design team supplied exactly three brand colors: #006B38 (deep green), #8DC63F (lime), #132A00 (near-black green). We model them as one Tailwind `primary` scale — lime at a light stop (~400), deep green at a mid stop (~700), near-black at 950 — with the remaining stops interpolated in OKLCH for consistency. The alternative (a separate `accent` scale for the lime, since it is a different hue family used for CTA/highlight roles) was considered and rejected: the team wants a single palette that literally contains only the three designed colors as real anchors. Consequence: there is no designed light tint of the deep green — light stops of the scale trend toward lime, and interpolated mid stops (500–600) are transitional olive-greens that should be used sparingly.
